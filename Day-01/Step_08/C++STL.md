# 📦 STL – CONCEPTS + PRACTICE PROBLEMS

---

## 1️⃣ CONCEPTS

- **Vector** – Dynamic array, supports indexing, push_back, pop_back  
- **Set** – Unique sorted elements, fast search  
- **Map** – Key-value pairs, keys unique, sorted by key  
- **Unordered Map (HashMap)** – Key-value pairs, keys unique, average O(1) lookup  

---

## 2️⃣ PRACTICE PROBLEMS

### Vector
1. Create a vector of n integers and print them.  
2. Reverse a vector.  
3. Sort a vector in ascending/descending order.  
4. Find maximum and minimum elements in a vector.  
5. Remove duplicates from a vector using `sort` + `unique`.  

### Set
6. Insert n elements into a set and print them (duplicates removed).  
7. Check if a number exists in a set.  
8. Find first element greater than or equal to X (`lower_bound`).  
9. Remove an element from a set.  
10. Count number of elements in a range.  

### Map
11. Count frequency of elements in an array.  
12. Find the most frequent element.  
13. Insert key-value pairs and print all keys in sorted order.  
14. Update a value for a given key.  
15. Remove a key-value pair.  

### Unordered Map
16. Count frequency of characters in a string.  
17. Check if a number exists as a key.  
18. Sum all values in the map.  
19. Merge two unordered maps.  
20. Implement a simple lookup table for n queries.
