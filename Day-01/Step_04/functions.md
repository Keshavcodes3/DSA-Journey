---

## 🌱 **Concept (Explained in Easy Language)**

A **function** is a small reusable block of work.

Think of a function like a **mini-machine**:
- You **give it input**  
- It **does a task**  
- It **gives output**

### Why use functions?
✔ Makes programs cleaner  
✔ Removes repetition  
✔ Helps break big problems into smaller ones  
✔ Easy to test and reuse

### Types of Functions (Simple)
1. **No input, no output** → Just does something  
2. **Input, no output** → Takes something, prints result  
3. **Input + Output** → Takes something, returns a value  
4. **Multiple inputs** → Useful for choices, validations, menus
