# 🔀 30 Conditional + Switch Case Flowchart Problems  
### From **Basic → Intermediate → Tricky**

These questions help beginners understand **decision-making**, **branching**, and **switch-case logic** using simple flowcharts.

---

## 🟢 **BASIC LEVEL (Simple Decision Making)**  
1. Input a number → Check if it is **even or odd**.  
2. Input age → Check if the person is a **minor** or **adult**.  
3. Input 2 numbers → Print the **greater** number.  
4. Input a number → Check if it is **positive**, **negative**, or **zero**.  
5. Input marks → Print **Grade A/B/C/Fail** based on ranges.  
6. Input a character → Check if it is a **vowel** or a **consonant**.  
7. Input a number → Check if it is **divisible by 5** and **divisible by 11**.  
8. Input a year → Check if it is a **leap year** or not.  
9. Input a number → Print if it is **single-digit**, **two-digit**, or **three-digit**.  
10. Input time in 24-hour format → Print **Good Morning / Afternoon / Evening / Night**.

---

## 🟡 **INTERMEDIATE LEVEL (Nested IF + Multiple Conditions)**  
11. Input three numbers → Print the **largest**.  
12. Input three angles → Check if they **form a triangle**.  
13. Input the sides of a triangle → Check if it is **scalene / isosceles / equilateral**.  
14. Input temperature → Classify as **Cold / Warm / Hot / Very Hot**.  
15. Input salary → Calculate **bonus** using conditional logic (e.g., 10%, 7%, 5%).  
16. Input a number → Check if it is **prime or not**.  
17. Input a number → Check if it is **Armstrong number** (e.g., 153).  
18. Input electricity units → Calculate **bill** based on slabs.  
19. Input a day number (1-7) → Print **Day name** using **switch-case**.  
20. Input month number → Print **number of days** using **switch-case**.

---

## 🔴 **TRICKY LEVEL (Logic-Based + Multiple Branching)**  
21. Input coordinates (x, y) → Check which **quadrant** the point lies in.  
22. Input a number → Check if it is a **Harshad (Niven) number**.  
23. Input amount → Calculate **discount** based on price ranges.  
24. Input a character → Classify as **Uppercase / Lowercase / Digit / Special Symbol**.  
25. Menu-driven calculator using **switch**: (+, -, ×, ÷).  
26. Input ATM withdrawal amount → Validate:  
    - Sufficient balance  
    - Multiple of 100  
    - Print new balance  
27. Input time (HH MM) → Convert to **12-hour format with AM/PM** using conditions.  
28. Input a number → Check if the **sum of digits is even or odd**.  
29. Input a date (dd-mm-yyyy) → Check if the date is **valid**.  
30. Mini Menu (Switch):  
      - 1 = Area of Circle  
      - 2 = Area of Rectangle  
      - 3 = Area of Triangle  
      - 4 = Area of Square
