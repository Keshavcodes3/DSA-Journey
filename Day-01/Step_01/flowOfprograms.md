## 🔰 **20 Super Beginner-Friendly Flowchart Problems**

### 🟦 Basic Understanding
1. Start → Print **"Hello World"** → End  
2. Input a number → Print the **same number**.

### ➕ Simple Math Logic
3. Input two numbers → Print their **sum**.  
4. Input two numbers → Print their **difference**.  
5. Input two numbers → Print their **product**.  
6. Input a number → Print whether it is **positive**, **negative**, or **zero**.

### 🔢 Number Conditions
7. Input a number → Check if it is **even or odd**.  
8. Input a year → Check if it is a **leap year**.  
9. Input marks → Print **Pass** or **Fail** (assume pass ≥ 40).  
10. Input a number → Print its **multiplication table up to 10**.

### 🔁 Loops & Repetition (Beginner Level)
11. Print numbers from **1 to 10**.  
12. Print the **sum of numbers from 1 to N**.  
13. Keep taking numbers until user enters **0** → Print **total sum**.  
14. Print **all even numbers** from 1 to N.  
15. Print **all odd numbers** from 1 to N.

### 📦 Basic Patterns & Logic
16. Input a number → Print how many **digits** it has.  
17. Input a number → Reverse the number (e.g., 123 → 321).  
18. Input a number → Check if it is a **palindrome**.  
19. Input a number → Print the **largest digit** in it.  
20. Input N → Print the **first N terms of the Fibonacci sequence**.

---

### 🌟 Tip for Beginners  
Start by identifying:  
- **Input**  
- **Process**  
- **Output**  

Then convert it into a simple flowchart.
