# 🔢 BASIC MATH & RECURSION – PROBLEMS

## Easy
1. Understand recursion by printing something N times  
2. Print name N times using recursion  
3. Print 1 to N using recursion  
4. Print N to 1 using recursion  
5. Sum of first N numbers  
6. Factorial of N numbers  
7. Fibonacci Number (simple recursion)  
8. Power of a number (a^b) using recursion  
9. Count digits in a number  
10. Sum of digits of a number  

## Medium
11. Product of digits of a number  
12. Reverse a number  
13. Check if a number is prime  
14. Find GCD of two numbers  
15. Find LCM of two numbers  
16. Sum of first N even/odd numbers  
17. Count trailing zeros in factorial of N  
18. N-th Fibonacci Number using memoization  
19. Solve Tower of Hanoi problem (N disks)  
20. Compute binomial coefficient C(n, k) recursively
