# 🛠️ DSA SETUP REQUIREMENTS – EXTENSIONS & TOOLS

This file lists all the **tools, VSCode extensions, and resources** you need to start your DSA journey efficiently.

---

## 1️⃣ IDE / Code Editors
- **VSCode** – Recommended IDE for beginners  
  - [Download VSCode](https://code.visualstudio.com/)  
- **Alternative IDEs**: CodeBlocks, CLion, IntelliJ IDEA (Java), PyCharm (Python)  

---

## 2️⃣ Compilers / Languages
- **C++ Compiler**
  - Windows: MinGW / WSL  
  - Linux: `sudo apt install g++`  
  - Mac: `brew install gcc`  
- **Java JDK** (if using Java)  
- **Python 3** (if using Python)  

---

## 3️⃣ VSCode Extensions
1. **Code Runner** – Run snippets and code quickly inside VSCode  
2. **tldraw** – Draw flowcharts, recursion trees, and diagrams  
3. **Markdown Preview Enhanced** – Preview your `.md` notes visually  
4. **LeetCode Extension** – Solve and submit problems directly from VSCode  
5. **GitLens** – Manage Git repositories effectively  
6. **Bracket Pair Colorizer / Rainbow Brackets** – Easily match brackets in code  
7. **GitHub Copilot (optional)** – Provides code hints and suggestions  
8. **C/C++ Extension (Microsoft)** – Required for debugging & intellisense  
9. **Python Extension (Microsoft)** – Required if using Python  
10. **Java Extension Pack** – Required if using Java
