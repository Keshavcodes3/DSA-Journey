# 🔁 Loop Questions (For → While → Mixed)

## 🌱 **Concept (Explained in Easy Language)**

A **loop** is used when you want to repeat something again and again.

### 🟦 **FOR LOOP (Repeat a known number of times)**
Use a **for loop** when you know:
- **How many times** the work needs to repeat  
Example: “Print numbers from 1 to 10” → You know it's exactly 10 times.

### 🟩 **WHILE LOOP (Repeat until a condition becomes false)**
Use a **while loop** when:
- You **don’t know** how many times it will run  
- It stops only when a condition becomes false  
Example: “Keep taking input until user enters 0”.

### 🟧 **DO-WHILE LOOP (Run at least once)**
Use a **do-while loop** when:
- The task must run **at least one time**, even if the condition is false  
Example: “Ask user for input at least once”.
